Notas sobre este Cronograma y Riesgos:
Flexibilidad del Cronograma: Este es un cronograma de alto nivel. La duración de las subtareas dentro de cada fase se ajustará. El desarrollo ágil o iterativo podría ser beneficioso, entregando valor en incrementos.
Feriados y Días Festivos en Paraguay: Para un cálculo preciso de días laborables, se deben descontar los feriados oficiales de Paraguay que caigan de lunes a viernes durante el período del proyecto. Este cronograma asume un flujo continuo de semanas laborables.
Algunos feriados importantes a considerar (las fechas pueden variar ligeramente o añadirse otros):
~12 de Junio (Paz del Chaco)
~15 de Agosto (Fundación de Asunción)
~29 de Septiembre (Victoria de Boquerón)
~8 de Diciembre (Virgen de Caacupé)
Navidad y Año Nuevo (impactarán la última semana de diciembre y primera de enero).
Estos feriados reducirán los días laborables efectivos y deberían considerarse al detallar el cronograma de tareas específicas. El buffer de 1 semana puede ayudar a absorber esto.
Dedicación de 8 horas/día, L-V: Es la base de trabajo.
Riesgos: La lista de riesgos es inicial y debe ser revisada y actualizada periódicamente.
Este plan ahora incluye una distribución temporal estimada para cada fase y una identificación inicial de riesgos con sus planes de mitigación.
